    Chatbot Project

Projeto fullstack de um chatbot, desenvolvido com Node.js no backend e React no frontend.

## Tecnologias
Frontend: React, Vite
Backend: Node.js, Express, CORS
Comunicação: API REST

## Estrutura
chatbot-project/
├── frontend/
└── backend/

## Fluxo

Usuário digita mensagem no React
React envia POST para API
API processa e responde
React renderiza a resposta

## Como rodar o backend

```bash
cd backend/chatbot-api
npm install
npm run dev
